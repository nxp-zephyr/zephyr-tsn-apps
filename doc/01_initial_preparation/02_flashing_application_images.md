# Flashing application images

i.MX RT1180 EVK board provides a USB connection (USB_OTG1) dedicated to flash programming.

i.MX RT1180 EVK board includes an integrated MCU-Link debug probe that supports the CMSIS-DAP protocol. This enables debugging, flash programming, and serial communication over a single USB connection.

Refer to the flashing procedures described in the [README.md](../../README.md#flashing-applications).
